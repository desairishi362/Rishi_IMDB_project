# Rishi IMDB Project

This repository serves as a comprehensive project for analyzing IMDB movie datasets and performing detailed analysis for insights and visualization. The dataset being referenced allows for exploring hundreds of movies along various metrics such as ratings, genres, revenue, and runtime, among others.

## Project Overview
This project includes:

1. **Dataset Overview:**
   - **File path:** `5-IMDB-Movie-Data.csv`
   - Contains information about movies, including attributes like Rank, Title, Genre, Runtime, Revenue, etc.
2. **Notebook Analysis:**
   - Exploratory Data Analysis (EDA) in a Jupyter Notebook (`Rishi_IMDB_project.ipynb`) to understand key trends.

## Features and Functionalities:
- Read and analyze the IMDB movie dataset.
- Extract useful insights such as top-rated movies by genre.
- Explore runtime distribution, revenue vs rating analysis, and much more.
- Visualizations are generated using tools like Matplotlib and Seaborn.

## How to Run
1. Clone the repository.
2. Ensure Python and the required libraries (`pandas`, `numpy`, `seaborn`, `matplotlib`) are installed.
3. Open the Jupyter Notebook (`Rishi_IMDB_project.ipynb`) and run the cells for analysis.

## File Contents
- **5-IMDB-Movie-Data.csv:** Dataset for analysis.
- **Rishi_IMDB_project.ipynb:** Notebook performing the analysis.

## Visualization Examples
Visualizations include:
- Bar graphs illustrating revenue across the top movies.
- Genre-specific insights.
